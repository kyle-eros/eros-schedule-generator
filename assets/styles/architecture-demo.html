Key Algorithms

VOSE ALIAS METHOD

O(1) weighted random selection from caption pool

Preprocessing: O(n) Selection: O(1)
algorithm 
# Weights: [0.4, 0.1, 0.3, 0.2]

# Build probability and alias tables
prob:  [1.0, 0.4, 1.0, 0.8]
alias: [ - ,  0 ,  - ,  2 ]

# Selection (constant time!)
1. Pick random index i
2. Pick random float r in [0,1]
3. if r < prob[i]: return i
   else: return alias[i]
FRESHNESS DECAY

14-day half-life exponential decay

freshness = 100 * (0.5 ^ (days / 14))

Minimum threshold: 30 (reached at ~24 days)
